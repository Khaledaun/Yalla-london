# Site Health

Generate a weighted health score (0-100) for a site using real platform data.

## Steps

1. Identify target site from arguments (default: yalla-london)
2. Use the `site-health` skill to gather data and compute score
3. Save snapshot to `docs/health-snapshots/` for trend tracking
4. Present iPhone-optimized results

## Arguments
$ARGUMENTS (site ID: yalla-london, arabaldives, french-riviera, istanbul, thailand, all)

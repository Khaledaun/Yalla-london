## 6. CANVA SETUP — STEP-BY-STEP

### Phase 1: Create Brand Kit
1. Go to **Canva → Brand Hub → Brand Kits**
2. Click **"+ Add new"** → Name it **"World Through My Eyes"**

### Phase 2: Add Brand Colors
Add these 9 colors in order:
```
#07A4F2  (Sky Blue — Primary)
#068DD0  (Blue — Secondary)
#03AD62  (Green)
#007965  (Dark Teal)
#85C342  (Lime)
#FFC417  (Gold/Yellow)
#9B1749  (Burgundy)
#000000  (Black)
#FFFFFF  (White)
```

### Phase 3: Upload & Configure Fonts
1. **Montserrat** — Already in Canva, add to brand kit as Body font
2. **Gilroy** — Already in Canva, add as Display font
3. **Rift Soft** — Upload via Brand Kit → Fonts → "Upload a font" (requires Pro)
4. **Shamel Family Sans** — Upload .ttf/.otf files (requires Pro)

If you don't have the font files, the original designer (phone: 00972599144950, "Good Time" studio) created the showcase and likely has the original .ai files with outlined fonts.

### Phase 4: Upload Logo Assets
1. Go to **Uploads** in Canva
2. Upload all logo PNGs from the extracted package
3. In Brand Kit, go to **Logos** section → Add uploaded logos
4. Set `WTME_Logo_Transparent.png` as the primary brand logo

### Phase 5: Upload Pattern Assets
1. Upload pattern tile PNGs to Canva Uploads
2. Optionally add to Brand Kit's **Graphics** section

### Phase 6: Create Template Designs
1. Import the original showcase PDF into Canva as a multi-page design
2. Extract individual poster layouts as templates
3. Save poster designs as brand templates for future reuse

## 7. VECTOR ASSETS — CRITICAL NOTE

The showcase PDF was created in **Adobe Illustrator CC 23.0**. The logo, patterns, and poster layouts are **vector graphics** in the PDF — not raster images. The PNGs I extracted are high-resolution rasterizations (600 DPI), which are excellent for digital use but not true vectors.

**For maximum precision, request from the original designer:**
- Logo as `.svg` or `.ai` file
- Pattern tile as `.svg`
- Font files (`.ttf` or `.otf`) for Rift Soft and Shamel Family Sans
- Original `.ai` poster templates

These vector files can be directly imported into Canva and will scale perfectly at any size.

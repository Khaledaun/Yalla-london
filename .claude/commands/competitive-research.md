# Competitive Research

Run a comprehensive competitive analysis using the Research Intelligence Agent.

## Steps

1. **Competitor Discovery** — Use `exa-search` to find top competitors per destination
2. **Content Inventory** — Use `firecrawl-scraper` to crawl competitor content libraries
3. **SEO Comparison** — Compare keyword rankings, domain authority, content depth
4. **Content Gap Analysis** — Identify topics competitors rank for that we don't cover
5. **Social Presence** — Use `tavily-web` to analyze competitor social strategies
6. **CRO Patterns** — Analyze competitor CTAs, forms, and conversion flows
7. **Positioning Analysis** — Use `marketing-strategy-pmm` for competitive battlecards
8. **Trend Detection** — Identify emerging trends competitors are capitalizing on

## Input

$ARGUMENTS — Specific competitor URL, destination, or "all" for full analysis

## Output

- Competitive landscape map per destination
- Content gap report with keyword opportunities
- Battlecards for top 3 competitors per site
- Social media strategy comparison
- 20 high-priority topics to create
- Recommended positioning adjustments
- Trend report with actionable opportunities
